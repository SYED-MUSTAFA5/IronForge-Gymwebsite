# 🏋️‍♂️ IronForge Gym

A modern, high-performance landing page for **IronForge Gym**. This project features a bold, fitness-driven UI designed to boost engagement and conversions.

---

## 🚀 Overview

IronForge is a fully responsive landing page built using **React** and **Tailwind CSS**. It delivers a clean, aggressive visual style tailored for the fitness industry, offering smooth scrolling and an optimized user experience across all devices.

---

## ✨ Key Features

- **Bold Hero Section** – High-impact visuals to instantly grab attention.  
- **Responsive Design** – Fully optimized for mobile, tablet, and desktop.  
- **Pricing Plans** – Clear and structured membership tiers for users.  
- **Testimonials** – Social-proof section to increase trust and conversions.  
- **Smooth Navigation** – Easy scrolling and intuitive layout.

---

## 🛠️ Tech Stack

- **Frontend:** React  
- **Styling:** Tailwind CSS  
- **Build Tool:** Vite  
- **Icons:** Lucide React / React Icons
