
# 🏋️‍♂️ IronForge-Gymwebsite

A modern, high-performance landing page for **IronForge Gym**. This project features a bold, fitness-oriented UI designed to drive user engagement and conversions.

---

## 🚀 Overview

IronForge is a responsive landing page built with **React** and **Tailwind CSS**. It focuses on a clean, aggressive aesthetic suited for the fitness industry, featuring smooth scrolling and a seamless user experience.

### ✨ Key Features
* **Bold Hero Section:** High-impact visuals to grab attention.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
* **Pricing Plans:** Clear and organized membership tiers.
* **Testimonials:** Social proof section for better conversion.
* **Smooth Navigation:** Easy-to-use scroll behavior.

---

## 🛠️ Tech Stack

* **Frontend:** [React](https://reactjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Icons:** [Lucide React](https://lucide.dev/) (or React Icons)
